Соглашение о применении cookies, кэша и локального хранилища

Редакция от 11.03.2026

1. Общие положения

Настоящее соглашение определяет порядок применения на сайте технических cookies, localStorage, sessionStorage и кэша браузера.

Указанные технологии используются только для корректной работы сайта и не применяются для рекламного таргетинга.

2. Какие данные сохраняются

  • Cookie `cookie_consent` - хранит факт согласия на использование cookies (до 365 дней).
  • sessionStorage `restore-scroll-y` - временно хранит позицию прокрутки до закрытия вкладки.
  • localStorage `site-theme` - хранит выбранную тему интерфейса до удаления данных сайта пользователем.
  • Кэш браузера - хранит копии статических файлов (CSS, JS, изображения) для ускорения загрузки страниц.

3. Цели использования

  • обеспечение входа в систему и корректной работы сессии;
  • сохранение пользовательских настроек интерфейса;
  • восстановление контекста страницы после отправки форм;
  • ускорение повторной загрузки сайта.

4. Согласие и его отзыв

Нажатие кнопки «Принять» в баннере означает согласие пользователя с настоящим соглашением.

Пользователь вправе отозвать согласие в любой момент, удалив cookies и данные сайта в настройках браузера. После удаления данных баннер будет показан повторно.

5. Ограничения обработки

Сайт не передает данные из cookies, localStorage и sessionStorage третьим лицам для рекламы. Обрабатываются только технические данные, необходимые для функционирования сервиса.
